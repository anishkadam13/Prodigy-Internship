This project was given by Prodigy Infotech as Task 02.


Description  
This project is a stopwatch web application built using HTML, CSS, and JavaScript.  
It allows users to start, stop, and reset the timer while displaying hours, minutes, and seconds accurately.

Languages used: HTML, CSS, JavaScript

File Structure  
index.html: The main HTML file for the stopwatch interface.  
style.css: Contains styling for layout and design.  
script.js: Contains JavaScript logic for stopwatch functionality.
