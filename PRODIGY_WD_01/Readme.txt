Description  
This project is a responsive photography landing page designed for a professional photography brand.  
It includes a visually rich hero section with a call-to-action, a services section showcasing different photography categories, and a “Why Choose Us” section highlighting key strengths.  
The design follows a premium black and gold theme with smooth hover effects and a clean layout suitable for portfolio and business use.

Languages used: HTML, CSS

File Structure  
index.html: The main HTML file for the webpage.  
style.css: Contains all the styling for layout, responsiveness, and hover effects.  
images/: Directory containing images used for the hero section, services, and other visual elements.
