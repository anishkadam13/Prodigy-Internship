Description  
This project is a Weather App built using HTML, CSS, and JavaScript.  
It fetches real-time weather data from an external API and displays temperature, weather condition, humidity, and wind speed for a searched city.

Languages used: HTML, CSS, JavaScript

File Structure  
index.html: Main HTML file for the weather app interface.  
style.css: Styling for layout and card-based UI.  
script.js: JavaScript logic for API fetching and data handling.

Note: Replace YOUR_API_KEY in script.js with your own OpenWeatherMap API key.
