Description  
This project is a simple and clean Tic Tac Toe web game built using HTML, CSS, and JavaScript.  
It allows two players to play turn by turn, detects winning and draw conditions, and includes a restart option.

Languages used: HTML, CSS, JavaScript

File Structure  
index.html: Main HTML file for the game interface.  
style.css: Styling for layout and UI.  
script.js: JavaScript logic for the game.
